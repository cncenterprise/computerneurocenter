Atmosfera parecida com visual novel / sistema fictício O estilo foi pensado para lembrar: * interfaces de jogos psicológicos * sistemas operacionais fictícios * desktops misteriosos * arquivos “recuperados” sem copiar diretamente nenhum jogo específico.
